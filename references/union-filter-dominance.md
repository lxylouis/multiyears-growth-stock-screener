# 并集筛选主导效应：条件A主导条件B

## 是什么

本筛选器的规则是 **A ∪ B**（OR 并集）：

```
保留 = (全期 ≥ cond_a_min)  ∪  (最新期 > cond_b_threshold)
```

实测发现，当 **条件A 较宽松**（如旧默认值 All ≥ 0.9）而条件B 较严格（如旧默认值 > 2.0）时，**条件A 是实际的主要筛子**，条件B 的阈值变化几乎不影响结果。

## 验证数据

同一批数据（5年×3期，旧默认值，仅改 cond_b_threshold）：

| 市场 | 指标 | 2.0x (14.87% CAGR) | 2.5x (20.11% CAGR) | 变化 |
|:---|---:|---:|---:|:---:|
| 标普500 | 双通过 | 271 | 271 | **±0** |
| | 营收通过 | 405/438 | 404/438 | **↓1** |
| 沪深300 | 双通过 | 127 | 127 | **±0** |
| | 营收通过 | 251/279 | 250/279 | **↓1** |
| 恒生指数 | 双通过 | 19 | 19 | **±0** |
| | 营收通过 | 79/88 | 79/88 | **±0** |

## 原因分析

A∪B 的通过数 = |A| + |B| - |A∩B|

- |A| 很大（绝大多数股票全期≥0.9）
- |B| 被 |A∪B| 消化——条件B 仅在**满足B但不满足A**的股票上产生增量
- 而一个股票"最新期翻2.5x" 通常也满足"全期≥0.9"（因为翻2.5x的5年中，各期都不太可能跌到0.9以下）
- 所以 B \ A 的结果集非常小，改变 B 的阈值几乎没有影响

## 推论

> 在 A∪B 筛选结构中，**主导筛子是条件A（较宽松的那个）**，条件B 只在 A 很严格或 B 远高于 A 时才有意义。

| 场景 | 实际效果 |
|:----|:--------|
| A 宽松, B 宽松 | 大多数都通过 → 无筛选力 |
| A 宽松, B 严格 | **A 主导筛选**，B 几乎无效 |
| A 严格, B 宽松 | **B 主导筛选**，A 几乎无效 |
| A 严格, B 严格 | 双条件严格筛选，通过率低 |
| A 和 B 互换（AND） | 完全相反——条件B 才是主要筛子 |

## 设计建议

如果希望 `cond_b_threshold` 真正发挥作用：

1. **收紧 cond_a_min**（如从 0.9 提到 1.0+）
2. **去掉条件A，只用条件B**（只看最新5年，不管历史）
3. 改用 **AND** 逻辑（前提是数据质量足够好，不会过度过滤）
